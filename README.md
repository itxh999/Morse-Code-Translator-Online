# Morse Code Translator Online

A professional, interactive, and high-fidelity **Morse Code Translator** designed for enthusiasts, students, and professionals. Translate text to Morse code and vice versa with real-time audio feedback and a realistic telegraph station experience.

## 🚀 Key Features

- **Bidirectional Translation**: Seamlessly convert English text to Morse code and decode Morse code back to English.
- **Realistic Audio Feedback**: High-quality sine wave oscillator providing authentic telegraph sounds.
- **Interactive Telegraph Station**: A dedicated "Telegraph Station" where you can manually tap out Morse code using your mouse or keyboard (Spacebar).
- **Real-time Visual Feedback**: Watch the translation happen character-by-character with synchronized highlighting and a blinking cursor.
- **Professional UI**: A dark-themed, modern interface built with Tailwind CSS, featuring smooth animations and responsive design.
- **Customizable Settings**: Adjust the frequency (Hz) and speed (WPM) of the Morse code audio to suit your preference.

## 🛠️ Tech Stack

- **React 18**: For a fast, component-based user interface.
- **Tailwind CSS**: For utility-first, responsive styling.
- **Framer Motion**: For smooth UI transitions and interactive animations.
- **Lucide React**: For clean and consistent iconography.
- **Web Audio API**: For low-latency, high-fidelity sound generation.

## 📖 How to Use

1. **Text to Morse**: Type your message in the input field to see the Morse code translation instantly. Click the "Play" button to hear the sequence.
2. **Morse to Text**: Switch tabs and type dots (.) and dashes (-) or use the **Telegraph Station** to manually input code.
3. **Telegraph Station**: Click the station at the bottom to activate keyboard control. Use the **Spacebar** to transmit dots and dashes.
4. **Settings**: Use the gear icon in the header to adjust audio frequency and playback speed.
